Install instructions:

Download the entire repository, and unzip. Copy paste the content folders into C:\Program Files (x86)\Steam\steamapps\common\Slay the Spire 2\mods (you will have to create the mods folder if it doesn't exist). After starting STS2, go to Settings -> Modding -> Mod Settings. Make sure the mods show up, you might have to check them and restart to enable?
